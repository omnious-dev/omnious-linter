## Installation

```bash
# NPM
$ npm install -D typescript tslint tslint-config-omnious tslint-react
# Yarn
$ yarn add -D typescript tslint tslint-config-omnious tslint-react
```

## Usage

```js
{
    "extends": "tslint-config-react",
    "rules": {
        // Additional, per-project rules...
    }
}
```
